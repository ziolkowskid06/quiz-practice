I want to create web app where i could exercise by doing various quizes. I want to put the app into the server and run on any device with internet access. 

At this time i don't need to any tools to create quizes from resources. We create only test quiz to check functionalities.



From technical tools i was thinking to use 

- Next.js (React framework)
- JavaScript
- CSS
- Github
- Vercel CLI



I want to display various quizes:

- true or false

- multiple choice

- select all that apply

- fill in the blank

- matching



In the future i want to have each quiz in separate .md. That will be uploaded to my server. Therefore the file shall contain the quiz and some metadata.



Once website opened i want to have a list of topics added and after selection i want to see types of the quizzes available.



Although the quizzes will be written in certain order, I want to to display them randomly every time time the quiz is launched. I mean mix questions and answers every time.  



During answering question i want to know the right answer once my answer is submitted. I want to see a total score once quiz is completed and possibility to look back to answered questions. The summary which one is false.
